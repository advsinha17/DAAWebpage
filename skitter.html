Dataset Description

The Autonomous systems by skitter dataset represents the connectivity of autonomous systems (AS) on the internet. Nodes of the network are ASes and if an AS i has a connection to AS j, the graph contains a directed edge from i to j. This dataset has 1,696,415 nodes and 11,095,298 edges. Distribution of Cliques and Runtime Comparison

The Bron-Kerbosch and CLIQUE (Tomita) algorithms were run on the as-skitter dataset. Due to the large size of the dataset, we were unable to run the Chiba algorithm on this dataset within reasonable time. The network has a total of 37,322,355 maximal cliques and the size of the largest maximal clique is 67 vertices.

The graph on the left above contains the runtimes of each algorithm for the dataset. The Bron-Kerbosch algorithm takes 213 seconds to execute, while the CLIQUE (Tomita) algorithm takes 360 seconds. The graph on the right shows the distribution of the maximal cliques of different sizes. There are 3.171609 million cliques of size 3, while there are only 4 maximal cliques of size 67.
