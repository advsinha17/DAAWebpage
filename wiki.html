Dataset Description

he Wikipedia vote network dataset captures votes from Wikipedia administrator elections. Nodes of the network are Wikipedia users and if a user i has voted for user j, the graph contains a directed edge from i to j. This dataset has 7,115 nodes and 103,689 edges. Distribution of Cliques and Runtime Comparison

All 3 algorithms were run on the Wikipedia vote dataset. The network has a total of 459,002 maximal cliques and the size of the largest maximal clique is 17 vertices.

The graph on the left above contains the runtimes of each algorithm for the dataset. The Bron-Kerbosch algorithm takes 1.51 seconds to execute, while the CLIQUE (Tomita) algorithm takes 2.39 seconds, and the Chiba algorithm takes 306 seconds.
