Dataset Description

The Netscience dataset contains scientific collaborations in network science. The nodes represent individual scientists (authors of papers) and an edge is present between 2 authors if they co-authored a paper together. There are 1589 nodes and 2742 edges present in this dataset. The details of the dataset are given in the table below.
